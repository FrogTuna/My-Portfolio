Project--2: ToolLibrary

This project is aiming to implement a Tool Library System based on c#. This system allows a registered member to view Tool collection from system and make a selection for tool that they want to borrow, once a member successfully borrowed one item, the system will record member’s borrowing items. Member also allow return borrowing item to system. On the other hand, system staff is allowing to add new a tool type and revise tool’s quality. It also could delete an existing tool type or edit tool” quality. In addition, as a user, user have to register our system and then access to authority of member’s section.

  • MainMenu:

    This is my main menu section. It has 3 different main functionalities, which including staff login, Member login and exit respectively. If we select invalid index number, it will invoke an alert until input a correct index number.
  • StaffMenu:

    Once user select this menu, it will be allowed use serveral functionalities below, which are creating, adding and deleting tool, In addition, it also allow user to register and delete member.
  • MemberMenu:

    This menu is designed for registered member, Once user have an account, then click MemberMenu and allow user to log in with thier accounts, Once they are successful to log in, they could use serveral functionalities below, which are borrowing, returning, viewing, displaing, checking tool.
